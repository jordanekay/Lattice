Lattice maps http:// and https:// URLs to custom native app URL schemes based on the host and path.

Supported Apps
--------------
* Tweetbot for Mac (twitter.com -> tweetbot://)
    * User Profiles
    * Statuses
* iTunes Store
* Mac App Store
* Dash (Apple documentation)
   
Future Plans
